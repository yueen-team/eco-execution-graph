# 张主任演示脚本

你们有法条,我们补现场；你们有法规知识库,我们补行业场景；你们有执法工具,我们补企业真实问题；这套图谱不是资料库,而是法条落地到现场的执行层。

## 阶段一:5 张精品开场

1. 危废包装容器标签信息不完整或与实物、台账不一致。
2. 危废包装容器“一物一码”与平台记录核查。
3. 危险废物管理台账记录不完整。
4. 危废出入库记录、电子标签二维码与转移联单不一致。
5. 危废暂存场所防渗、防漏及泄漏收集措施不完善。

这 5 张的叙事顺序是:现场标签 -> 一物一码 -> 台账记录 -> 转移闭环 -> 暂存实体风险。

## 阶段二:危废专题全量目录

开场 5 张讲价值,随后切到危废全量目录,说明当前所有危废相关候选都已经形成执行卡切片。目录中分为主任开场精品、专题扩展切片、合并展示切片和规模化候补切片。合并展示卡只证明覆盖,不单独讲,避免重复和夸大。

## 演示边界

- 不说“已认定违法”,统一说“建议核查、建议完善、存在管理风险”。
- 合并展示卡只在目录中出现,不单独当主任主线讲。
- 暂不演示 full 版云南踩雷地图和月报对比,等待真实聚合数据与 ETO 盲评。
