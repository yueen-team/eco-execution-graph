# ADR-0004 现场事件为一等内容源,公开标准退为骨架

- 状态:Accepted(2026-06-10)
- 来源:2026-06-01 审计意见 #1(炸裂级第一条)

## 决策

图谱内容的**主来源**是 EcoCheck `semantic_event_outbox`(现场蒸馏 v2 事件:问题/证据/整改/成效/ESO-ETO delta)。公开标准与法规(eco-kb baselines、HJ 标准、系数手册)是**本体骨架与挂载点**,不是内容主体。

## 理由

公开标准花钱能买、谁都能下,不构成壁垒;每天 100 家企业产生的现场判断流是抄不走的。审计已证实:纯自上而下编码的库会变成"治理精良却没住人的样板间"(spl 快照仅 ~6 节点)。

## 后果

- `pipeline/ingest.py`(outbox 消费)优先级高于 `import_baseline.py`(骨架导入)的内容扩展。
- 衡量健康度的指标是"来自现场事件的边占比",不是节点总数。
- 现场事件入图默认 CANDIDATE,晋级走人工审核(治理膜保留,审计强项不丢)。
