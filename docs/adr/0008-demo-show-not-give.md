# ADR-0008 演示"看得见、带不走"

- 状态:Accepted(2026-06-10)

## 决策

政府侧演示**必须展示私有层**(证据标准、整改闭环、报告表达、踩雷蒸馏过程)——震撼恰恰来自这些;但展示 ≠ 交付:

1. 演示运行在我方设备/账号,不留安装包、不留数据副本;
2. 演示数据中企业一律为合成样本(结构真实、标识虚构);
3. 任何交付物(共有包、缺口报告、卡片共有视图)走 tier=shared 过滤;证据类别与概念级字段要求可展示,证据判断标准/真实样例/审核笔记/整改模板全文/报告表达模板全文只显示"已建立标准(N 条)"占位;
4. 演示叙事按 C 刀法:先放对方执法工具的"有法条无场景"现状,再现场展示接上图谱后的样子。

## 理由

成功标准是"震惊和撼动"+ 后续背书,震撼源于私有层的存在和活性(蒸馏计数器);但私有层是商业壁垒,一旦交出无法收回。"看得见带不走"是两个目标的唯一交集。

## 后果

- graph-ui 需要"演示模式"开关:全量视图(内部)/ 共有视图(可交付)一键切换——切换本身就是演示的一部分(向主任展示授权边界的工程化,这也是专业度信号)。
