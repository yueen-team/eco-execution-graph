# ADR-0009 ego 邻域视图,不做全图毛线球

- 状态:Accepted(2026-06-10)

## 决策

graph-ui 的默认与主要交互是 **ego 视图**:以单节点(法条/问题/行业/企业类型)为中心,展开 1-2 跳邻居,按边类型筛选、按 confidence 着色、节点带 tier 徽章。不提供"全图力导向铺开"作为主视图(最多作为 3 秒开场动画)。

## 理由

全图力导向 demo 很酷、实战没用(GitNexus 美学的正确吸收方式是交互范式,不是毛线球)。ego 视图同时是**上下文装配的调试器**:运行时给 AI 装配上下文 = 取节点邻域子图,ego 视图让"AI 这次加载了哪个子图"可视、可审——这是第二刀(月报回灌)的工程价值,不只是演示美学。

## 后果

- UI 状态模型围绕"当前中心节点 + 展开深度 + 边类型过滤器"设计。
- 选 Cytoscape.js 而非 react-force-graph:对类型化边标签、复合节点、样式映射支持更好,适合精选小图;炫酷大图不是目标。
