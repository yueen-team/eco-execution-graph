# RAG Doc Ref Registry

`rag_doc_ref` values are external references only. They must not contain full law or technical-standard text.

Current namespaces:

- `tencent-lke://law/...`
- `tencent-lke://spec/...`
- `tencent-lke://eco-kb/...`

The registry is materialized in `reports/rag-citation-resolution-report.json`.
